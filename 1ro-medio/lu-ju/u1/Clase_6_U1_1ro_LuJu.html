Unidad 1 — The World of Technical Work

📘 OA9 · OA2 ⏱ 90 minutos 🧭 Material de clase

🎯 Objetivo de la clase

Comprender un texto instructivo sobre el armado de un circuito electrónico simple, identificando vocabulario de componentes y conectores de secuencia.

Class focus: active vocabulary + guided practice + exit ticket.
🟢 Inicio (15 min)
  • Mostrar imágenes (o componentes reales si se tienen) de: resistor, capacitor, LED, transistor, circuit board, soldering iron.
  • Preguntar: "Can you guess what each one does? Have you seen these before?"
  • Breve comparación: "Last class we talked about electricity. Electronics is different. How?" (Electricidad = energía; Electrónica = información/control).
🔵 Desarrollo (60 min)
