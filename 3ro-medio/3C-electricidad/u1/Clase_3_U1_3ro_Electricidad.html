📖 Grammar Focus: Simple Present (Daily Routines)

We use the Simple Present to describe actions that happen every day or regularly.

  • I / You / We / They: Use the normal verb. (We install solar panels)
  • He / She / It: Add -s or -es to the verb. (He measures the voltage)
  • Negative: I don't wire... / He doesn't wire...
