📖 Activity 1 — Case Study: The Color Mismatch (25 min)

The Color Mismatch

A print shop receives an urgent order: 5,000 brochures for a hotel chain. The client's brand color is a specific shade of dark blue. The job must be perfect.

Step 1: The technician opens the digital file and checks the color settings. The file is in RGB mode instead of CMYK. This is the first problem — RGB colors look different when printed.

Step 2: She converts the file to CMYK and makes a color proof (test print). The blue looks too purple compared to the client's sample.

Step 3: She adjusts the CMYK values manually: she reduces the magenta and increases the cyan until the color matches the sample. She prints another proof.

Step 4: The second proof matches the client's sample. The client approves it. The technician sets up the printing press and runs the full order, checking the first 20 sheets carefully.

Solution: The wrong color mode (RGB vs CMYK) and incorrect color values caused the mismatch. Converting and adjusting the CMYK values fixed the problem.
